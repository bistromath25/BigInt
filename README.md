# BigInt
A simple Big Integer (BigInt) class for C++

# Features
* Read BigInts and write BigInts
* Assign BigInt values using =
* Create BigInts from strings or long long values
* Addition
* Subtraction
* Compare BigInts using <, >, <=, >=, ==, and !=
